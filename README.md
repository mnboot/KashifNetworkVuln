## Contribution

For now, the project only allow specific AASU students to contribute <br/> If you are one of them:

1. Fork and make branch and name it based on your last or first name “kashifnetwork-yourname”
2. add folder based on your role(e.g. if your role to make wrapper name folder “wrapper” then add as much classes as
   needed in it
3. since you can’t run your classes to test the class
    1. open test folder in `src/test/java`
    2. add folder and name same as on step 2 (e.g. `src/test/java/wrapper/`)
    3. add java class file (e.g. `WeapperTest.java`)
    4. use [JUnit](https://en.wikipedia.org/wiki/JUnit) testing framework
4. use [Javadoc](https://www.oracle.com/technical-resources/articles/java/javadoc-tool.html#format) for each method
   especially when that methods need to be used by others.
5. Obey coding and OOP principals; for example, private attributes with getter-setter, each method should do one task,
   allow polymorphism, etc.
6. Never use internal/default access modifier
